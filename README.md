# FanCraftPearls

一个有趣的Minecraft Bukkit插件，改变了末影珍珠的功能，让玩家可以骑乘末影珍珠飞行，而不是传统的传送。

## 功能特点

- **骑乘飞行**：玩家丢出末影珍珠后会骑在上面，而不是被传送
- **连续飞行**：在飞行过程中可以继续使用末影珍珠来改变方向
- **无落地伤害**：取消末影珍珠的落地伤害
- **无冷却时间**：取消末影珍珠的冷却时间
- **自动清理**：丢出新末影珍珠时，旧的末影珍珠会自动删除

## 兼容性

- Minecraft 1.21
- Paper 服务器

## 安装方法

1. 下载最新版本的 FanCraftPearls.jar 文件
2. 将其放入服务器的 `plugins` 目录
3. 重启服务器或使用 `/reload` 命令重载插件

## 使用方法

1. 拿起末影珍珠
2. 像平时一样右键丢出
3. 你会自动骑上末影珍珠并开始飞行
4. 移动鼠标改变飞行方向
5. 再次右键丢出新的末影珍珠可以改变飞行方向

## 配置

目前本插件无需配置，安装即可使用。

## 技术细节

- 插件会拦截末影珍珠的默认行为
- 创建新的末影珍珠实体并将玩家设为乘客
- 持续更新末影珍珠的飞行方向，使其跟随玩家视角
- 每个玩家的末影珍珠独立管理，不会互相干扰
